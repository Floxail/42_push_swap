# Push_swap

## Description 

A project that teaches us how to create a complex sorting algorithm with special instructions. We have to sort a list of numbers with a limited instruction objective, so that we learn how to optimize. 

## instruction

### Compilation

make

### Execution

./push_swap [list of integers]
